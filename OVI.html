OBJETIVO

Fundamentar teóricamente a los estudiantes sobre las generalidades básicas del Diseño Web para que puedan utilizar de forma correcta los lenguajes HTML5 y CSS3, aplicarlos en la construcción de Sitios Web e implementarlos a través de sistemas de control de versiones. Así mismo Ofrecer información para que el estudiante pueda operar y administrar sistemas de control de versiones como GIT o GITHUB.

OBJETIVOS ESPECIFICOS

Objetivo específico 1:
Brindar pautas para que le estudiante pueda evaluar y desarrollar Sitios Web que satisfagan las necesidades del usuario y dar recomendaciones concretas para su actualización.

Objetivo específico 2:
Conocer el uso de GIT/GITHUB, HTML5 y CSS3 y su estructura, para que así lo ponga en práctica para el desarrollo de sitios web.
